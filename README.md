# LikeC4 Architecture Visualization

This project contains a LikeC4 architecture visualization deployed on GitHub Pages.

## 🚀 Live Demo

Visit the live demo at: `https://scrptkiddo.github.io/parser/`

## 📁 Project Structure

- `index.html` - Main HTML file
- `assets/` - Static assets (CSS, JS, fonts, icons)
- `likec4-views.js` - LikeC4 views configuration
- `.github/workflows/deploy.yml` - GitHub Actions workflow for automatic deployment

## 🛠️ Development

The source architecture files are located in the `vacsparser-likec4` folder. To make changes:

1. Edit the architecture files in `../vacsparser-likec4/src/`
2. Run `npm run build` in the `vacsparser-likec4` directory
3. Copy the built files to this directory
4. Commit and push to trigger automatic deployment

## 📦 Built with LikeC4

This project is built using [LikeC4](https://likec4.dev/), a modern tool for creating architecture diagrams as code.
